# **App Name**: OfertaHub

## Core Features:

- Intelligent Copywriting Tool: Generate persuasive marketing messages for your offers using generative AI tailored to specific audience niches.
- Command Center Dashboard: A clean SaaS-style dashboard displaying real-time metrics for link clicks and offer engagement levels.
- Daily Offers Management: A robust data table to view, filter, and curate daily deals before distribution.
- Sent History Database: Review a complete chronological log of all messages and offers sent through the system to specific groups.
- Niche-Based Segmenting: Manage multiple distribution groups organized by product niches to ensure targeted marketing.
- Evolution API Integration: Automated messaging integration using the Evolution API to sync directly with communication platforms.
- Brand Personalization Hub: Customize the interface with unique logos, default message structures, and brand color presets.

## Style Guidelines:

- Primary color: Modern Royal Blue (#4461E1) for a reliable, professional software feel.
- Background: Soft Blue-Grey Tint (#F5F6FA) to reduce eye strain in management settings.
- Accent color: Bright Cerulean (#1FA2E1) used for high-contrast interactive elements and call-to-actions.
- Font pairing: 'Space Grotesk' for clean, technical headlines and 'Inter' for highly readable dashboard body text and tables.
- Sleek, monolinear icons with subtle shadows to emphasize the automation and efficiency theme.
- Fixed-side sidebar navigation with a responsive cards-and-tables workspace layout typical of professional SaaS automation tools.
- Smooth hover states and subtle fade-in transitions for metric updates and side-panel switches.
